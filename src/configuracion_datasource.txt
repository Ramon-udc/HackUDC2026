============================================================
           PROYECTO: PEIXEALERT (REPLICABLE)
============================================================

Este proyecto monitoriza las condiciones marítimas de Galicia 
y predice capturas de pesca mediante la integración de la 
API de MeteoGalicia, Grafana y un Bot de Telegram.

------------------------------------------------------------
1. ESTRUCTURA DE ARCHIVOS
------------------------------------------------------------
- PeixeGalicia.py: Motor del bot en Python (Lógica biológica).
- PeixeAlert-1772454823137.json: Configuración del Dashboard.
- CONFIGURACION.txt: Este manual técnico.

------------------------------------------------------------
2. CONFIGURACIÓN DEL DATA SOURCE EN GRAFANA
------------------------------------------------------------
Para que los paneles funcionen, se debe crear un Data Source 
con los siguientes parámetros exactos:

- Tipo: JSON API (Plugin de Marcus Olsson)
- URL: https://servizos.meteogalicia.gal/apiv5/getNumericForecastInfo
- Auth: No Authentication
- Headers:
    - Accept: application/json
- Additional Settings:
    - Query string: limit=100
