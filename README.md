# HackUDC2026

**Descripción**

Nuestro proyecto busca mostrar la información sobre el clima (altura de las olas, temperatura y velocidad) en algunos puertos de galicia

## Tecnologías Utilizadas

Python: Lenguaje principal para el desarrollo de la aplicación.

Grafana cloud: entorno en el que se muestran los datos recogidos

Meteogalicia: fuente de la información


## Características Principales

Interfaz simple e intuitiva para la interacción del usuario.

Gráficación clara
